# 🚦 Traffic Accident Data Analysis

## 📌 Project Overview
This project focuses on cleaning, processing, and visualizing traffic accident data to extract meaningful insights.

---

## 🧠 Key Features
- Data cleaning (missing values, duplicates, outliers)
- Exploratory Data Analysis (EDA) using Python
- Interactive dashboard using Power BI

---

## 📊 Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook
- Power BI

---

## 📈 Key Insights
- Peak accidents occur during evening hours
- Majority crashes are non-injury cases
- Driver behavior is a major cause
- High percentage of vehicle damage observed

---



## 🎯 Outcome
This project demonstrates data cleaning, visualization, and storytelling skills.
